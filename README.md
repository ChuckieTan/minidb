# minidb
A mini database for study

## Compiling
You need:
- Cmake build system
on Linux/OS X/FreeBSD
```bash
$ git clone https://github.com/ChuckieTan/minidb.git
$ cd minidb
$ mkdir build/ && cd build/
$ cmake -DCMAKE_BUILD_TYPE=Release ..
$ make -j
```

## Running
just run minidb_release directly
```bash
./minidb_release
```
